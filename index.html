Drawer

make it drawerable.

把任意 macOS app 变成从屏幕边缘滑入/滑出的抽屉 —— 一个快捷键,秒呼秒收。高频但不必常驻的 app(AI 助手、聊天、终端),用完即走,不打断手头的事。

下载最新版 ↓

要求 macOS 14.0+,通用二进制(Apple Silicon 与 Intel)。首次打开未公证可能被 Gatekeeper 拦:右键 App → 打开。

功能

  • 边缘抽屉:左/右/上/下,快捷键滑入滑出
  • 多抽屉:各绑快捷键;同边互斥、异边独立
  • 失焦自动收起、开机自启
  • 内置自动更新(Sparkle),支持 Beta 频道

用法

  • 菜单栏 Drawer 图标 → 设置 → ＋ 选 app → 选边缘 → 录快捷键
  • 首次需在 系统设置 → 隐私与安全性 → 辅助功能 勾选 Drawer
